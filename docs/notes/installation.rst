Installation
================================


From pip
------------------------------------
You can install or upgrade langml/langml-cli from pip:

.. code-block:: bash

   pip install -U langml



From Github
------------------------------------
You can also install the latest langml/langml-cli from Github:

.. code-block:: bash

    git clone https://github.com/4AI/langml.git
    cd langml
    python setup.py install
